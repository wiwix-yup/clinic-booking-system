## Описание проекта
Система для записи пациентов на приём к врачу с проверкой занятости. Исключает двойную запись одного врача на одно и то же время.

## Технологии
- Python 3.11+
- SQLite
- pytest

Туториал на запуск кода: 
скачайте пайчарм и при загрузке поставьте эту галочку.
<img width="803" height="571" alt="image" src="https://github.com/user-attachments/assets/686fa27e-1cdb-497d-b793-11585b97c303" />


поочередно скачайте архивированные файлы: sozdanie_bd, add_data, proverka-zanyatosti, test_overlap.
после переместите файлы в папку пайчарм, она должна выглядеть примерно так: 

<img width="498" height="510" alt="снимок директории" src="https://github.com/user-attachments/assets/e33590f7-58f5-47c4-88c5-b4884c333867" />

В ходе разработки и тестировки были найдены все баги и вовремя исправлены. Все функции работают исправно, выполняется полный обязатяльный функционал записей с созданием бд и внесением данных.
