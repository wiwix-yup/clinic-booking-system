# clinic-booking-system
Система записи на приём к врачу в частной клинике
# Медицинский навигатор: запись к специалисту

## Команда
- Шацкова София – разработчик бэкенда, интеграция БД
- Анастасия Попова – разработка интерфейса, тестирование

## О проекте
Система позволяет пациентам частной клиники записываться на приём к врачу с проверкой свободных слотов. Исключается двойная запись. Врачи, дата, время – гибко настраиваются.

## Технологии
- Python 3.11
- Flask (микрофреймворк)
- SQLite3
- pytest для тестов
- HTML/CSS (Bootstrap 5)
