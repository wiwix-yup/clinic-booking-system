# Проект: Запись к врачу в частной клинике

## Команда
- Шацкова София-разработчик
- Попова Анастасия-тестировщик

## Описание проекта
Система для записи пациентов на приём к врачу с проверкой занятости. Исключает двойную запись одного врача на одно и то же время.

## Технологии
- Python 3.11+
- SQLite
- pytest
